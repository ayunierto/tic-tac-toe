# Tic Tac Toe — versión simple

Un juego muy simple de Tic Tac Toe hecho con HTML/CSS/JS. Para ejecutarlo, abre `index.html` en tu navegador.

Pasos:

1. Abrir el archivo `index.html` (doble clic o arrastrar al navegador).
2. Jugar haciendo clic en las casillas. Reiniciar con el botón "Reiniciar".

Mejoras opcionales: marcador, IA simple, interfaz responsive, animaciones.
